# Blackjack Count Trainer

Visual Hi-Lo card counting trainer. Configure decks, players, and rounds — watch realistic basic-strategy play unfold, then guess the running and true count.

## Live site

After enabling GitHub Pages (Settings → Pages → Source: GitHub Actions), the app is available at `https://<your-username>.github.io/blackjack-trainer/`.

## How it works

1. Set number of decks (1–8), players (1–7), and rounds (1–30)
2. Watch each round: players follow basic strategy, dealer plays to 17+
3. After all rounds, enter your running count and true count
4. See how close you were with a round-by-round breakdown

Toggle **Show Hi-Lo Values** during practice to see +1/0/−1 badges on every card.
